## Description  
**Free Shipping Progress** is a lightweight, easy‑to‑use WordPress plugin that displays a dynamic progress indicator showing customers the remaining amount needed to reach your store’s free shipping threshold.  
Perfect for WooCommerce shops that want to increase cart value and improve customer experience.

You can customize colors, text, threshold amounts, alignment, and display the progress anywhere using a simple shortcode.

## Features  
- Display remaining amount to reach free shipping  
- Fully customizable progress color  
- Customizable text fields:  
  - Remaining amount text  
  - Free shipping activated text  
- Set your free shipping amount  
- Choose text alignment  
- Shortcode support  
- Lightweight and 100% free

## Shortcode

```
[free_shipping_progress]
```

## Settings  
The plugin includes a simple settings page where you can configure:

- Progress Color (Color Picker)  
- Remaining Text  
- Free Shipping Activated Text  
- Free Shipping Amount  
- Text Align (left, center, right)

## Installation  
1. Download the plugin and upload it to `/wp-content/plugins/`.  
2. Activate the plugin through the *Plugins* menu in WordPress.  
3. Go to **Settings → Free Shipping Progress** to configure options.  
4. Add the shortcode `[free_shipping_progress]` anywhere you want the progress bar to appear.
